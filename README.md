# TrafficStudyGo

TrafficStudyGo is an **educational and research-focused Go project**
for understanding **network traffic behavior** and **defensive responses**
such as rate limiting and blocking.

This repository may include historical traffic-generation code
for **controlled, permission-based environments**, along with
offline educational analysis tools.
